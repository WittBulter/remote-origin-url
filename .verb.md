## Usage

Using the defaults:

```js
var originUrl = require('remote-origin-url');
originUrl()
// "https://github.com/jonschlinkert/remote-origin-url.git"
```

Specify the `cwd` to use:

```js
origin.url(__dirname);
```


## Run tests

Install dev dependencies:

```bash
npm i -d && npm test
```
